#node-express-mongo-ejs的ToDoList

It's a todolist.

- [node](https://nodejs.org/zh-cn/)
- [express](expressjs.com.cn)
- [mongo](http://www.mongoosejs.net/)
- [ejs](https://ejs.bootcss.com/)
- `body-parser` for post

run

`node app`

then open
`
http://localhost:3000/todo
`
